## 📋 Descripción

Portfolio profesional de **Francisco Charriel Romero**, Desarrollador Junior y Técnico SMR especializado en Desarrollo de Aplicaciones Multiplataforma. El sitio presenta de forma clara mi perfil, formación, experiencia y stack tecnológico.

## 🗂️ Estructura del proyecto

```
fcharriel.github.io/
├── index.html       # Página principal del portfolio
├── estilo.css       # Estilos y diseño visual
├── animacion.js     # Animaciones y efectos de scroll
├── perfil.png       # Foto de perfil
└── CV.pdf           # Currículum vitae descargable
```

## ✨ Secciones

| Sección | Descripción |
|---|---|
| **Inicio** | Presentación, foto, botones de contacto y descarga de CV |
| **Sobre Mí & Formación** | Perfil profesional, datos personales y titulaciones (DAM, SMR, especializaciones) |
| **Experiencia** | Línea de tiempo con los puestos desempeñados |
| **Stack Tecnológico** | Herramientas y lenguajes: Java, Android, HTML5, CSS3, PHP, MySQL, Linux, Git |

## 🛠️ Tecnologías utilizadas

- **HTML5** — Estructura semántica
- **CSS3 (Vanilla)** — Diseño con modo oscuro, gradientes y sistema de grid tipo Bento
- **JavaScript** — Animaciones reveal al hacer scroll
- **Font Awesome 6** — Iconografía
- **Devicons** — Iconos de tecnologías
- **Google Fonts** — Tipografías *Poppins* e *Inter*

## 🚀 Despliegue

El sitio se despliega automáticamente mediante **GitHub Pages** al hacer push a la rama `main`.  
No requiere build ni dependencias externas.
